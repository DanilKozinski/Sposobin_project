Сайт доступен по ссылке: https://omg-sposobin.vercel.app

Этот проект представляет собой конспект избранных глав книг по элементарная теория музыки трёх разных авторов. Сайт сделан для демонстрации умения работы с базой данных SQL и созданием веб-приложений с помощью FLask.


На данном сайте есть четыре страницы- основная(с полным списком всех терминов из всех книг) и отдельная страница для каждого автора. 


Функционал сайта:
- Навигация по содержанию: слева можно увидеть оглавление- на каждый термин можно нажать и перейти на соответсвующий раздел текста на той же странице
- Поиск по странице: При написании текста в соответствующей панели сверху и нажатии кнопки "Найти" выведутся соответсвующие элементы БД, содержащие этот текст в описании или названии.
- Переход на другие страницы
