# Конспект по теории музыки

Веб-приложение на Flask с конспектами трёх авторов по элементарной теории музыки.

**Ссылка на проект:** [https://omg-sposobin.vercel.app](https://omg-sposobin.vercel.app)

## Функционал
*   **4 страницы:** Главная (все термины) + по странице для каждого автора.
*   **Интерактивное оглавление** для быстрой навигации по странице.
*   **Поиск** по названию и описанию терминов в БД.
*   **Навигация** между разделами сайта.

## Технологии
*   Backend: **Flask (Python)**
*   Database: **SQLite**
*   Frontend: **HTML, CSS**
*   Deployment: **Vercel**
